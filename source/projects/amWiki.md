amWiki 致力于让大家可以更简单、更便捷的建设个人和团队文库。  
使用 amWiki，可已创建无限级目录，无服务端仍可全库内容搜索，接口文档和接口测试一次搞定，(除了接口测试)完美运行于 Github Pages...

amWiki 可以分别在 Atom 编辑器或 Nodejs-Npm 命令行两个独立的平台运行，功能基本相同，所创建的文档可以无缝交叉使用
