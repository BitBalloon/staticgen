* Hidden Grids
* Multiple Grids per Page
* Grid in a Modal
* CellClass
* Add/Delete/Swap Data
* Adding and removing columns
* Adding a row header
* HeaderCellClass
* Fonts and Installation
* Tooltips
* RTL Support
* Grid Menu
* Accessibility
* Large Dataset
* Horizontal Scrolling
