ag-Grid is the first high performance and feature rich data grid for AngularJS. Light weight, minimal footprint, no external dependencies.

#### Features:

* Sorting
* Filtering
* Resizing
* Pinning
* Selection
* Value Getters
* Cell Styling
* Cell Rendering
* Cell Templates
* Editing Cells
* Context
* Refresh
* Grouping Headers
* Header Rendering
